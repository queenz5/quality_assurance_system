# quality_assurance_system
一个 AI 驱动的集需求分析和用例生成质量保障系统，为开发团队提供从需求到测试的全流程质量保障解决方案。  ## 核心功能 - AI 测试用例生成 ：基于需求文档自动生成结构化测试用例 - 层级化测试管理 ：模块→子模块→测试用例的层级展示，支持完整 CRUD 操作 - 数据概览仪表盘 ：实时统计缺陷密度、需求覆盖率和 BUG 趋势 - 结构化步骤管理 ：完整的测试步骤管理（序号、操作、输入、预期结果） - 统一 YAML 存储 ：确保数据一致性和可维护性 ## 技术栈 - 后端：FastAPI + Python 3.10 - 前端：React + Material-UI - 数据：YAML/JSON 文件系统 - AI：集成大语言模型
