## 🚀 启动指南

### 1. 启动后端（终端1）

```bash
cd qa_system
./start.sh

# 访问 API 文档: http://localhost:8000/docs
```

### 2. 启动前端（终端2）

```bash
cd frontend
./start.sh

# 访问前端: http://localhost:3000
```

### 3. 使用系统

1. 打开浏览器访问 http://localhost:3000
2. 左侧导航栏切换不同功能
3. 查看数据仪表盘了解整体质量
4. 使用AI辅助测试生成用例或分析BUG
5. 查看智能报告了解项目质量

## 🎨 界面特性

### 响应式设计
- **桌面端 (>960px)**: 固定侧边栏，完整布局
- **平板端 (600-960px)**: 自适应网格布局
- **移动端 (<600px)**: 抽屉式导航，单列布局
