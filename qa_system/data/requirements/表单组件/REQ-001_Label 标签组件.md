# REQ-001: Label 标签组件

## 基本信息
- **模块**: 表单组件
- **子模块**: Label 标签组件
- **优先级**: 高
- **状态**: 待开发
- **创建时间**: 2026-04-13

## 需求描述
为小程序提供内置的 Label 标签组件，用于改进表单组件的可用性。点击标签区域时，会自动触发其关联的表单控件，从而扩大表单控件的可点击热区，提升用户交互体验。

## 功能点
1. 支持通过 for 属性绑定控件
   - 业务规则: for 属性值为目标控件的 id，点击 label 时触发该 id 对应的控件; 仅限绑定 button、checkbox、radio、switch、input
   - 约束条件: 仅限绑定 button、checkbox、radio、switch、input
2. 支持嵌套控件
   - 业务规则: 直接将表单控件放置在 label 标签内部; 若嵌套多个控件，点击时仅触发第一个
   - 约束条件: 若嵌套多个控件，点击时仅触发第一个
3. 支持优先级处理
   - 业务规则: 当同时使用 for 属性和内部嵌套时，for 属性优先级更高; for 绑定的控件与嵌套控件冲突时，以 for 为准
   - 约束条件: for 绑定的控件与嵌套控件冲突时，以 for 为准

## 前置条件
- 小程序基础库版本 >= 1.0.0
- 支持 WebView 和 Skyline 渲染框架

## 后置条件
- 点击 label 区域成功触发关联的表单控件
- 表单控件的状态正确更新（如 checkbox 勾选、radio 选中等）

## 业务规则
| 规则编号 | 规则描述 | 优先级 |
|---------|---------|--------|
| BR-001 | for 属性优先级高于内部嵌套 | 高 |
| BR-002 | 仅支持绑定 button、checkbox、radio、switch、input | 高 |
| BR-003 | 嵌套多个控件时，仅触发第一个 | 中 |
| BR-004 | 点击热区扩大至整个 label 区域 | 高 |

## 异常处理
- **for 值对应的控件不存在**: 点击 label 无反应
- **for 值对应的控件类型不支持**: 点击 label 无反应
- **嵌套控件类型不支持**: 控件正常渲染，但点击 label 不触发该控件
- **同时使用 for 和嵌套且冲突**: 优先响应 for 绑定的控件
