# REQ-002: Progress 进度条组件

## 基本信息
- **模块**: 基础组件
- **子模块**: Progress 进度条组件
- **优先级**: 高
- **状态**: 待开发
- **创建时间**: 2026-04-13

## 需求描述
为小程序提供内置的 Progress 进度条组件，用于可视化展示操作或任务的完成进度。该组件应支持自定义进度百分比、显示进度信息、自定义样式（颜色、圆角、线宽、字体）、动画效果及动画完成事件监听。

## 功能点
1. 支持设置进度百分比
   - 业务规则: percent 取值范围 0~100，非必填
   - 约束条件: 超出范围的值按边界处理（<0 按 0，>100 按 100）
2. 支持显示进度信息
   - 业务规则: show-info 为 boolean，默认 false，开启后在进度条右侧显示百分比
   - 约束条件: 字体大小可通过 font-size 属性自定义
3. 支持自定义样式
   - 业务规则: 支持圆角（border-radius）、线宽（stroke-width）、进度条颜色（activeColor）、背景颜色（backgroundColor）、字体大小（font-size）
   - 约束条件: 基础库 2.3.1+ 支持 border-radius 和 font-size
4. 支持进度条动画
   - 业务规则: active 为 boolean，默认 false，开启后进度条从左往右播放动画
   - 约束条件: 需同时设置 active=true 才会触发动画
5. 支持动画模式控制
   - 业务规则: active-mode 支持 backwards（从头播放）和 forwards（从上次结束点续播）
   - 约束条件: 基础库 1.7.0+ 支持
6. 支持动画速度控制
   - 业务规则: duration 表示进度增加 1% 所需毫秒数，默认 30
   - 约束条件: 基础库 2.8.2+ 支持
7. 支持动画完成事件
   - 业务规则: bindactiveend 事件在动画完成时触发
   - 约束条件: 基础库 2.4.1+ 支持

## 前置条件
- 小程序基础库版本 >= 1.0.0
- 使用 WebView 渲染框架

## 后置条件
- 进度条正确渲染在页面指定位置
- 进度百分比、样式、动画效果符合开发者设置
- 动画完成时正确触发 bindactiveend 事件

## 业务规则
| 规则编号 | 规则描述 | 优先级 |

| BR-002 | show-info 默认 false，开启后右侧显示百分比 | 高 |

| BR-008 | duration 默认 30ms/1% | 中 |

## 异常处理
- **percent < 0**: 按 0 处理
- **percent > 100**: 按 100 处理
- **color 属性使用**: 控制台输出警告"请使用 activeColor 替代 color"
- **active-mode 值不合法**: 按默认值 backwards 处理
- **duration <= 0**: 按默认值 30ms 处理
