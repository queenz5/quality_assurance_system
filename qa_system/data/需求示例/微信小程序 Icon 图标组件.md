## 需求描述
为小程序提供内置的 Icon 图标组件，用于在页面中快速展示系统预设的标准化图标，无需额外引入图片资源。该组件应支持多种图标类型、自定义大小和颜色，并兼容不同基础库版本和渲染框架。

## 功能点
1. 支持 9 种内置图标类型
   - 业务规则: type 为必填项，仅支持 success、success_no_circle、info、warn、waiting、cancel、download、search、clear
   - 约束条件: 不支持自定义图标，如需自定义需使用 image 组件
2. 支持自定义图标大小
   - 业务规则: 默认 23px，支持数字（默认 px）和带单位的字符串
   - 约束条件: 基础库 2.4.0+ 支持 rpx/px 等单位，2.21.3+ 支持 rem 等单位
3. 支持自定义图标颜色
   - 业务规则: 颜色取值与 CSS color 属性语法一致
   - 约束条件: 支持十六进制、RGB、RGBA、颜色名称等 CSS 标准格式

## 前置条件
- 小程序基础库版本 >= 1.0.0
- 使用 WebView 渲染框架（Skyline 渲染引擎以官方最新适配为准）

## 后置条件
- 图标正确渲染在页面指定位置
- 图标大小和颜色符合开发者设置

## 业务规则
| 规则编号 | 规则描述 | 优先级 |
|---------|---------|--------|
| BR-001 | type 属性为必填项，仅支持 9 种内置图标 | 高 |
| BR-002 | size 属性默认值为 23，默认单位为 px | 高 |
| BR-003 | 基础库 2.4.0+ 支持带单位的 size 字符串（rpx/px） | 中 |
| BR-004 | 基础库 2.21.3+ 支持 rem 等其他单位 | 中 |
| BR-005 | color 属性支持 CSS color 标准语法 | 高 |

## 异常处理
- **type 未填写**: 组件不渲染任何内容，控制台输出警告
- **type 值不合法**: 组件不渲染，控制台提示"不支持的 icon type"
- **size 为负数**: 按默认值 23px 处理
- **color 格式错误**: 按浏览器默认颜色处理或忽略
